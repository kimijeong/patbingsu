# 환자 표정/자세 이미지

앱(`index.html`)은 대화 상황에 맞춰 이 폴더의 이미지를 자동으로 바꿔 보여줍니다.
**이미지가 없으면 기존 SVG 캐릭터로 자동 폴백**되므로, 아래 파일을 채워 넣기만 하면 됩니다.

## 폴더 구조

```
assets/patient/
├── male/      ← 남성 환자(증례 성별='남')일 때 사용
└── female/    ← 여성 환자(증례 성별='여')일 때 사용
```

`male/` 와 `female/` 각각에 **아래 8개 파일**을 같은 이름으로 넣으세요(확장자 `.png`).

| 파일명          | 언제 표시되나                         | 준비할 레퍼런스 |
|-----------------|---------------------------------------|------------------|
| `smile.png`     | 평소(기본 상태)                       | 웃음             |
| `angry.png`     | 통증을 호소할 때                      | 화남(찡그림)     |
| `sad.png`       | 메스꺼움/구역을 호소할 때             | 슬픔             |
| `think.png`     | 걱정·고민할 때                        | 생각             |
| `surprise.png`  | (예비) 놀랄 때                        | 놀람             |
| `question.png`  | 환자가 되물을 때(질문, "...?")        | 질문             |
| `lie_back.png`  | 검진(눕기) 시 — 바로 누운 자세        | 뒤로 누움(천장 보기) |
| `lie_front.png` | (예비) 엎드린 자세                    | 앞으로 누움      |

> 합계: `male/` 8장 + `female/` 8장 = 16장.

## 상황 ↔ 이미지 매핑 (현재 기본값)

- 기본/평소 → `smile`
- 통증 호소 → `angry`
- 메스꺼움/구역 → `sad`
- 걱정/고민 → `think`
- 환자가 질문(되물음) → `question`
- 검진 위해 눕힘(검진맵의 🛏 눕기, "누우세요" 등) → `lie_back`

매핑을 바꾸고 싶으면 `index.html` 의 `EMO_IMG` 객체를 수정하면 됩니다.

## 권장 사항

- 배경은 단색(또는 투명 PNG) 권장 — 패널 위에 깔끔하게 얹힙니다.
- 인물 크기/구도를 표정 8장끼리 비슷하게 맞추면 전환이 자연스럽습니다.
- 세로 인물은 약 600×880px, 누운 자세는 가로로 넓은 비율을 권장합니다.
